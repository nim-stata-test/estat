4.4 Multi-Objective Pareto Optimization

Methodology

Used NSGA-II algorithm to find Pareto-optimal heating configurations:

  • Algorithm: NSGA-II (Non-dominated Sorting Genetic Algorithm II)
  • Population: 100 individuals
  • Generations: 100
  • Evaluations: 10,000
  • Pareto solutions: 100

Decision Variables

VariableRangeDescription
setpoint_comfort[19.0, 22.0] °CComfort mode target
setpoint_eco[12.0, 19.0] °CEco mode target (12°C = frost protection)
comfort_start[06:00, 12:00]Comfort period start
comfort_end[16:00, 22:00]Comfort period end
curve_rise[0.80, 1.20]Heating curve slope

Objectives

  1. Average Temperature: Maximize mean T_weighted during 08:00-22:00 (higher is better)
  2. Grid import: Minimize total kWh purchased from grid
  3. Net cost: Minimize grid cost - feed-in revenue (CHF)

Constraint (Soft Penalty)

Low-temperature violation: T_weighted < 18.5°C for no more than 20% of daytime hours (08:00-22:00). Solutions exceeding this threshold are penalized but not excluded.
